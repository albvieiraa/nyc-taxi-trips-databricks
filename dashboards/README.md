# Dashboards

Este diretório contém artefatos de visualização do projeto.

## KPI Dashboard
Dashboard criado no Databricks utilizando tabelas da camada Gold.

### Principais métricas exibidas:
- Total de corridas
- Receita total
- Ticket médio
- Receita por dia
- Receita por hora
- Análise por tipo de pagamento

> Observação: o dashboard foi criado diretamente no Databricks.  
> Este repositório contém capturas de tela para fins de documentação e portfólio.
